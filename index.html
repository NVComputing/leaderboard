NVComputing Leaderboard Processor

How it works:
1) Paste your current leaderboard JSON from GitHub (with a top-level users array) below.
2) Copy/paste Attendance / POTW / HW lines from the google sheets
3) Click "Update Leaderboard" to increment counts or create missing users (defaults to 0; attendance/potw/hw set to 1 on first creation).
4) ACSL/USACO are manually set in the collapsed section and auto-populate from the input JSON.
5) Paste and commit the output JSON back onto GitHub.
